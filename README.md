# SenSante
Medical pre-diagnosis assistant for Senegal.

## Description
SenSante uses Machine Learning to assist in the pre-diagnosis of common diseases (malaria, influenza, typhoid) based on patient symptoms.

## Project Structure
- `data/`     : Patient data (CSV)
- `models/`   : Serialized ML model
- `api/`      : FastAPI API
- `frontend/` : Web interface
- `notebooks/`: Exploration scripts
